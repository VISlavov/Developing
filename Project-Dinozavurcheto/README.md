Requirements:

	- ruby version 2.1.0
	
	- rubygems
	
	- gem rubygame (Make sure you install all dependencies - https://github.com/rubygame/rubygame/wiki/Install)
	
	- gem rubygoo

	- gem rubytrackselector

Template:

	cd source
	
	ruby run.rb

Example:


	ruby run.rb

Tips:

	In-game controls will not work if your keyboard layout is not english.
